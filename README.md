# Foundations of Computational Interaction (the proposed model)

**Foundations of Computational Interaction (the proposed model)** is an interdisciplinary scientific synthesis and research program for formally describing, measuring, modeling, and predicting human-computer interaction.

---

## Repository Structure

This repository is organized into three main layers:

1.  **[docs/](file:///Users/zachshallbetter/Projects/the proposed model/docs/) (Conceptual Corpus & Knowledge System of Record)**
    *   `01_Knowledge_System_of_Record/` — Canonical definitions, glossary, relational ontology (v0.3), and contract definitions.
    *   `02_Research_Papers/` — Scholarly papers and arguments organized in logical dependency order.
    *   `03_External_Literature/` — Supporting external literature.
    *   `04_Reference/` — Reference bibliography (v1.1.2), dataset schemas, templates, and concept graphs.
    *   `05_Publication/` — Publication guidelines and consistency audits.
    *   `90_Project/` — Release manifests, audits, and previous zip packages.
2.  **[app/](file:///Users/zachshallbetter/Projects/the proposed model/app/) (C Reference Substrate)**
    *   A freestanding C Reference Substrate implementation demonstrating computational coherence for a deterministic subset of the proposed model ontology.
    *   Includes validation schemas, benchmark programs, and stress tests.
3.  **[_archive/](file:///Users/zachshallbetter/Projects/the proposed model/_archive/) (Archived Releases)**
    *   Preserved historical release data and pre-normalization files.
