# Outstanding Editorial Issues

## Repository boundary

The attached archive contains the publication-infrastructure repository, not the complete Knowledge System of Record and research-paper corpus. Corpus-wide definition and paper-heading verification therefore cannot be completed from this archive alone.

## Bibliography

Authority validation, duplicate resolution, and missing metadata remain separate bibliographic-verification work.

## Figures

Only one reusable canonical figure is present. Source figures, editable originals, captions, accessibility text, and rights status remain incomplete.

## DOCX fields

Generated DOCX files use static headings and text. Dynamic Word cross-reference fields, equation sequences, and indexes require the final publication assembly.
