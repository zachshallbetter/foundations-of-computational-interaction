# Reference Quality Report

## Strengths

- Every canonical record has a stable citation key.
- Exact duplicate records were merged conservatively.
- Existing identifiers were syntax- or checksum-checked.
- Missing fields have explicit unresolved reasons.
- BibTeX and CSL JSON exports are synchronized.

## Weaknesses

- Many source records encode publisher or journal details only inside free-text notes.
- Several titles include container information as part of the title.
- Author strings use mixed BibTeX and prose conventions.
- Most records have not been matched to authoritative registry records.
- ORCID data is absent and should not be backfilled by name matching without manual confirmation.

## Rating

Internal research use: acceptable. External scholarly publication: conditional on authority verification of cited references actually used in the release manuscript.
