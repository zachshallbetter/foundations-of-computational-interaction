Authoritative metadata policy

Crossref REST API documentation: https://www.crossref.org/documentation/retrieve-metadata/rest-api/
Crossref Simple Text Query: https://www.crossref.org/documentation/retrieve-metadata/simple-text-query/
Library of Congress MARC 21 bibliographic format: https://www.loc.gov/marc/bibliographic/

Crossref states that its public REST API exposes metadata deposited by publishers and trusted sources, including bibliographic fields, post-publication updates, ORCID identifiers, and Retraction Watch data. The Simple Text Query accepts up to 1,000 references for DOI matching. MARC 21 provides field-level bibliographic description conventions.

Runtime limitation

The available web interface exposed documentation and selected search results but did not permit automated registry API calls or form submission from the artifact runtime. Accordingly, individual fields are marked authority-verified only when an authoritative record or surfaced publisher record was available. All other fields remain explicitly unresolved or internally present but externally unverified. No metadata was inferred.
