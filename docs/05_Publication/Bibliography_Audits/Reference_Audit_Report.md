# Reference Audit Report

## Scope

All 312 source BibTeX records were parsed, normalized, and evaluated. Exact duplicates were grouped by normalized title and year. The richest record was retained without changing theoretical content.

## Results

- Original records: 312
- Canonical records: 267
- Exact duplicate records merged: 45
- Records with DOI: 14
- DOIs with valid syntax: 14
- Malformed DOIs: 0
- Records with ISBN: 4
- ISBNs with valid checksum: 4
- Records with volume: 52
- Records with issue: 52
- Records with pages: 107
- Individually externally corroborated records: 2

## Verification classes

- Authority-verified: supported by an authoritative publisher or registry record.
- Externally corroborated: supported by an external bibliographic result, but not a machine-retrieved registry record.
- Syntax/checksum valid, authority unresolved: identifier format is valid but the authoritative record was not retrieved.
- Unresolved: no authoritative exact match was established.

## Finding

The bibliography is normalized and auditable. It is not fully authority-complete because most entries lack registry-verified metadata. Unresolved fields are retained as blanks and enumerated in Missing_Metadata_Report.csv.
