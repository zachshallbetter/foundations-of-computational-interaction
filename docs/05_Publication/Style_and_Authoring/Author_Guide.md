# Author Guide

Use canonical terms and identifiers from the glossary and ontology. Import definitions by reference; do not redefine them. Separate observations, measurements, inferences, and conclusions. Identify every contract used. State boundaries, timescale, population, task, projection, context, uncertainty, and falsification conditions. Cite primary sources where possible and cite the canonical corpus for project-specific synthesis. Record proposed theoretical changes only as Version 2.0 candidates.

## Citation style

Use one citation key per work from the master bibliography. Do not create local variants. Verify identifiers before publication.

## Figures and tables

Assign stable IDs, captions, source, version, and concept links. Provide editable source plus PDF/PNG exports.
