# Appendix A - Candidate Version 2.0 Changes

The following items were discovered during integration and were not incorporated into Version 1.0:

1. Evaluate whether "Foundations of Computational Interaction" remains the preferred program name after empirical validation; the current synthesis supports an interdisciplinary research program rather than disciplinary autonomy.
2. Consider formally ratifying the distinction between broad ordinary-language interaction and the strict interaction-episode research definition.
3. Evaluate whether transfer should become a first-class ontology node in a future normative revision. It is currently represented primarily through contracts and research papers.

These are proposals for later review, not changes to the Version 1.0 theory.
