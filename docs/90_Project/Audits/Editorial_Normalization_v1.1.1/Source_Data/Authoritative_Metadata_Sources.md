# Authoritative Metadata Sources

- Crossref REST API: https://www.crossref.org/documentation/retrieve-metadata/rest-api/
- Crossref metadata best practices: https://www.crossref.org/documentation/principles-practices/best-practices/
- DataCite Metadata Schema: https://schema.datacite.org/
- ORCID Integration Guide: https://info.orcid.org/documentation/integration-guide/

These sources define the recommended authority-validation path. The present build validates identifier syntax, ISBN checksums, duplicates, and completeness locally. It does not claim authoritative API matching for every unresolved reference.
